Welcome to the MagAOX Catalog Tool

To use the web tool, please fill out all the parameters below (object name(s), observation date, rotator mode, rotator offset, and data to display) and click on the "submit" button.

If observing more than one object, please enter all objects' names, separated by commas (eg. "Vega,Sirius,GQ Lupi,HD43691"). Any SIMBAD-resolvable name can be used for each object.

Observation date can be entered through a calendar widget, accessible through a dropdown arrow on the right-hand side of the date entry box.

Data for observations on a given day will range from sunset on that day to sunrise on the next day.

For more information on Rotator Mode and Offset, please click here.

This web tool can generate:
  • a Magellan-formatted catalog entry for all object
  • a spectral energy distribution plot for each object
  • an airmass plot for all objects
  • a visibility statistics plot for all objects

Please select all the data data you want generated. The web tool will always generate the catalog entry.
